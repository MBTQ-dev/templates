API Documentation

Available Endpoints

  • GET /nodes - Retrieve all nodes in the graph
  • GET /edges - Retrieve all edges (connections) in the graph
  • GET /graph - Retrieve complete graph structure
  • POST /query - Execute custom graph queries

Response Format

All responses are returned in JSON format with the following structure:

{
  "status": "success",
  "data": { ... },
  "timestamp": "2024-01-01T00:00:00Z"
}
